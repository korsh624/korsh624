Соревнования состоятся по номинациям:

18.01.2022 - Hello, Robot! Lego - соревнования для обучающихся 2-6 классов на базе образовательных конструкторов LEGO Mindstorms и LEGO Spike.

  • РобоИсполнитель - соревнования для обучающихся 2-4 классов;
  • РобоКвест - соревнования для обучающихся 5-6 классов;

Дополнительные материалы:

  • Общие положения соревнований "Hello, Robot! Lego"
  • Макет поля РобоИсполнитель

19.01.2022 - Hello, Robot! Open - соревнования для обучающихся 3-7 классов на базе любых образовательных конструкторов, кроме LEGO.

  • РобоКурьер - соревнования для обучающихся 3-5 классов;
  • РобоПеревозчик - соревнования для обучающихся 6-7 классов;

Дополнительные материалы:

  • Общие положения соревнований "Hello, Robot! Open"
  • Макет поля РобоКурьер
  • Макет поля РобоПеревозчик

20.01.2022 - Региональный этап Олимпиады школьников "Робофест" - практикоориентированная олимпиада по физике для обучающихся 7-11 классов и студентов СПО, включающая теоретическую и практическую часть.

Теоретический тур отборочного и финального этапа является индивидуальным конкурсом школьников по общеобразовательному предмету олимпиады - физике. На региональном этапе будет проводиться на образовательной платформе sdo.robofest.ru по потокам. На финальном этапе будет проводиться в очном (дистанционном при неблагоприятной эпидобстановке) формате в ФГБОУ ВО МГУ им М.В. Ломоносова. Участники выполняют задания по физике в форме письменной работы. Примеры заданий и ссылки на материалы для подготовки доступны на официальном сайте олимпиады.

Практический тур олимпиады - соревнования "Робокарусель". Для участия в соревнованиях команде необходимо подготовить трех роботов для выполнения трех различных заданий:

  • Робобой
  • РобоДомВверхДном
  • РобоШахматы

Дополнительные материалы:
